# Tomáš Němec

Statická osobní vizitka Tomáše Němce pro doménu `tomasbtc.cz`.

## Struktura

- `index.html` - hlavní stránka
- `domeny.html` - přehled tematických domén
- `style.css` - vzhled webu
- `script.js` - mobilní menu a rok ve footeru
- `assets/` - logo, favicon a sdílecí obrázek
- `CNAME` - vlastní doména pro GitHub Pages

## Lokální otevření

Nejjednodušší kontrola:

1. Otevři složku projektu.
2. Dvakrát klikni na `index.html`.
3. V prohlížeči zkontroluj hlavní stránku i odkaz `Domény`.

## GitHub Pages

Po nahrání do GitHubu zapni Pages z větve `main` a složky `/root`.
V nastavení Pages nastav vlastní doménu `tomasbtc.cz`.
